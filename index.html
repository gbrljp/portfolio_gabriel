SOBRE MIM

Meu nome é Gabriel Passos, tenho 21 anos e sou natural da cidade de São Paulo. Atualmente, sou estudante da área de Tecnologia da Informação (TI), campo pelo qual sou profundamente apaixonado e motivado a contribuir.
Desde a minha infância, sempre tive um interesse natural por tecnologia. Durante a minha juventude, passei grande parte do meu tempo explorando videogames, programas de computador e jogos no Game Boy. No entanto, foi apenas aos 16 anos que comecei a entender a vastidão e as possibilidades da área de TI. Esse interesse inicial se transformou em uma verdadeira paixão quando fui contratado como Jovem Aprendiz na área de Tecnologia da Informação.
Essa oportunidade me proporcionou um conhecimento mais aprofundado sobre os diferentes segmentos que um profissional de TI pode atuar, ampliando minha visão sobre a diversidade de caminhos e especializações dentro do campo.

Cursos

Designer de Interface WEB

Concluí o curso de Designer de Interface Web com uma carga horária de 60 horas, no qual adquiri conhecimentos essenciais para o desenvolvimento e design de interfaces digitais. Durante o curso, trabalhei com as principais ferramentas da área, incluindo HTML5, JavaScript e CSS, para a criação de páginas web dinâmicas e responsivas. Como parte do aprendizado prático, desenvolvi meu próprio site, aplicando os conceitos adquiridos em design, usabilidade e performance. Este curso me proporcionou uma sólida compreensão das boas práticas de desenvolvimento front-end, preparando-me para criar interfaces atraentes e funcionais, alinhadas às necessidades dos usuários e às tendências tecnológicas atuais.

BANCO DE DADOS PARA DATA SCIENCE

Este curso abordou os fundamentos e práticas essenciais de bancos de dados aplicados à ciência de dados. Entre os tópicos tratados estão o design e modelagem de banco de dados, linguagem SQL para manipulação e consulta de dados, técnicas de integração com ferramentas de análise de dados e práticas de otimização de consultas. O treinamento enfatizou a aplicação de bancos de dados em projetos de Data Science, incluindo análise de grandes volumes de dados, visualização e extração de insights

PYTHON - AVANÇADO

Concluí o curso de Python Avançado com uma carga horária de 40 horas, no qual aprofundei meus conhecimentos em programação com Python, explorando suas funcionalidades e bibliotecas mais complexas. Durante o curso, estudei técnicas avançadas de programação, como programação orientada a objetos, manipulação de arquivos, integração com bancos de dados, e criação de scripts automatizados. Além disso, adquiri experiência com bibliotecas populares, como Pandas, NumPy e Matplotlib, para análise e visualização de dados. O curso me proporcionou uma compreensão profunda das boas práticas de codificação e otimização de performance, além de me capacitar a resolver problemas complexos de maneira eficiente e estruturada.

EXCEL 2016 AVANÇADO

Este curso avançado aprofundou o domínio das funcionalidades do Excel 2016, abordando ferramentas como tabelas e gráficos dinâmicos, funções complexas para análise de dados, uso de validação de dados, consolidação de informações e cenários de projeção. O treinamento foi voltado para aprimorar a eficiência no processamento e análise de grandes volumes de dados, oferecendo suporte à tomada de decisões estratégicas.

POWER BI

Este curso aprofundado explorou o uso do Power BI como ferramenta de Business Intelligence, capacitando os participantes a transformar dados brutos em insights visuais e acionáveis. Abrangeu conceitos de modelagem de dados, criação de dashboards interativos, integração com diferentes fontes de dados e aplicação de DAX (Data Analysis Expressions) para cálculos avançados. Durante o treinamento, foram desenvolvidas habilidades práticas para geração de relatórios dinâmicos e visualizações estratégicas voltadas à tomada de decisões em diversos contextos organizacionais.

JAVASCRIPT - FUNDAMENTOS

Este curso introdutório abordou os fundamentos do JavaScript, capacitando os participantes a desenvolver aplicações web dinâmicas. Os tópicos incluíram a estrutura da linguagem, manipulação de elementos DOM, controle de eventos, e introdução à programação assíncrona com funções como setTimeout e fetch. Durante o treinamento, foram desenvolvidos projetos práticos que permitiram aplicar conceitos básicos de lógica de programação e interação com páginas web.
